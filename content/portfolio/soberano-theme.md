## Overview

Tema Hugo construido do zero com foco em privacidade, performance e configurabilidade. Dark-only, monospace, zero dependencias externas.

## Funcionalidades

- Sistema de privacidade com 3 presets e 21 flags granulares
- Sovereignty features: warrant canary, Tor onion-location, NIP-05 Nostr, PGP
- Portfolio section com status, stack e links
- Busca client-side com Pagefind
- JSON-LD schema completo (Person, BlogPosting, CreativeWork)
- Pure CSS, zero frameworks JavaScript

## Resultados

- Lighthouse 100/100 em todas as categorias
- Build time < 500ms para sites com 100+ paginas
- Zero requests externos em modo paranoid
